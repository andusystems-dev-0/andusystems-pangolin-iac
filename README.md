This repo will be used for migrating my Pangolin server instance from Hostinger to AWS to better support a full AWS/On-Prem environment. Adding AWS functionality to my deployment for redundancy, resiliency, and future server locale enhancements so that I can better locate where my services are being served from. For now using us-east-1 and local (us-east) but future enhancements will allow me to serve resources from a machine closer to the client for lower response times. I will continue to serve clients on the east coast of the US via local servers but am looking at experimenting with AWS edge/cache functionality for faster response times.
